# Checklist do Orquestrador

## Antes de Iniciar o Ciclo
- [ ] context/empresa.md está atualizado
- [ ] context/posicionamento.md está atualizado
- [ ] context/servicos.md está atualizado
- [ ] context/publico-alvo.md está atualizado
- [ ] context/concorrencia.md está atualizado
- [ ] context/restricoes.md está atualizado
- [ ] Objetivo do ciclo definido e documentado
- [ ] Outputs do ciclo anterior revisados

## Durante o Planejamento
- [ ] Agentes necessários identificados
- [ ] Ordem de execução definida
- [ ] Insumos de cada agente listados
- [ ] Dependências entre agentes mapeadas
- [ ] Critérios de sucesso definidos
- [ ] Cronograma aprovado

## Antes de Acionar Cada Agente
- [ ] task.json criado para o agente
- [ ] Insumos necessários disponíveis
- [ ] Output esperado documentado

## Ao Final do Ciclo
- [ ] Todos os agentes executados
- [ ] Outputs salvos em outputs/[ano-mes]/
- [ ] Resultados comparados com critérios de sucesso
- [ ] Aprendizados documentados para o próximo ciclo
- [ ] context/ atualizado com novas informações
